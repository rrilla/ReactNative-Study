# 인스타그램 클론 코딩
전예홍, 『Do it! 리액트 네이티브 앱 프로그래밍』을 보고 Study!

## 설치 방법
MAC:

```sh
npm install
npx react-native link react-native-vector-icons
npx pod-install
```

윈도우:

```sh
npm install
npx react-native link react-native-vector-icons
```
